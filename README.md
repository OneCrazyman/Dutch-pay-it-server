#Dutch Pay It

- 카메라로 영수증 사진을 찍어서 메뉴/수량별 더치페이 기능
- 영수증의 메뉴/항목을 구별하기 위해 ML 오픈소스 활용
- 더치페이를 하기 전에 정산내역서를 사용자에게 보여주는 기능
- 홈화면, 식당명 및 구성원 입력창 & 사진 인식 확인, 정산내역서 화면
